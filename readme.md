# 🐄 Cowtype

**Cowtype** is a small CLI tool inspired by `cowsay`, but it prints messages with a **typewriter animation effect**.
---

## ✨ Commands
- --help or -h: Prints this list
- --delay=value or -d=value: Changes the amount of time each character takes to be typed
- --file or -f: Opens the specified file and types it out
- --cowfile=value or -c=value: Changes the cowfile

---

## 📦 Installation

```bash
git clone https://github.com/HaibaoM55/cowtype
cd cowtype
make
```
